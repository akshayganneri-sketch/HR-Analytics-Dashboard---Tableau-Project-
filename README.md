## 📌 Project Overview

This project presents an interactive HR Analytics Dashboard built using Tableau to analyze employee attrition, workforce demographics, and job satisfaction trends.

---

## 🎯 Business Problem

Organizations struggle to understand why employees leave and how workforce factors impact retention.

---

## 📊 Solution

Developed a dynamic Tableau dashboard to:

* Track employee attrition
* Analyze department-wise trends
* Understand demographic patterns
* Identify key drivers of employee turnover

---

## 🛠️ Tools Used

* Tableau
* Excel / CSV

---

## 📈 Key Insights

* Higher attrition observed in specific departments
* Salary and job role significantly impact retention
* Younger employees show higher turnover rates
